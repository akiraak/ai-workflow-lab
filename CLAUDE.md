# CLAUDE.md

このファイルは Claude Code がリポジトリを扱う際のガイドラインです。

## プロジェクト概要

AIを普段の開発・作業ワークフローに組み込む手法を調査・検証するリポジトリ。

## 言語・コミュニケーション

- ユーザーとのやり取りは日本語で行う
- コミットメッセージは日本語で書く
- コード中のコメントは英語でも日本語でも可（文脈に合わせる）

## コーディング規約

- 検証コードであっても、再現性を意識して書く
- 依存関係がある場合は requirements.txt や package.json で管理する
- 実験ごとにディレクトリを分けて整理する

## タスク管理

- **TODO.md** — 作業予定・進行中のタスクを `- [ ] タスク内容` 形式で管理する
- **DONE.md** — 完了したタスクを `- [x] タスク内容 (完了日: YYYY-MM-DD)` 形式で記録する
- タスク完了時は TODO.md から DONE.md へ移動する

## Git 運用

- ブランチ: main ブランチで直接作業（個人実験リポジトリのため）
- コミット: 変更の意図がわかるメッセージを付ける
