# ai-workflow-lab

AIを普段の開発・作業ワークフローに組み込む手法を調査・検証するためのリポジトリです。

## 目的

- AIツール（Claude Code、GitHub Copilot など）を活用した開発ワークフローの実験
- プロンプトエンジニアリングやエージェント活用のベストプラクティスの蓄積
- 自動化・効率化パターンの検証

## リポジトリ構成

```
ai-workflow-lab/
├── README.md        # このファイル
├── CLAUDE.md        # Claude Code 向けのプロジェクト設定
├── TODO.md          # 作業予定・進行中のタスク
├── DONE.md          # 完了したタスクの記録
└── LICENSE
```

## 使い方

このリポジトリでは、各テーマごとにディレクトリやスクリプトを作成して検証を行います。
検証結果や知見は適宜ドキュメントとして残していきます。
